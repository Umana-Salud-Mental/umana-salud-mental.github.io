# Umana - Plataforma de Salud Mental

Sitio web oficial de Umana, una plataforma digital dedicada a hacer la salud mental accesible para todos.

## 🌐 Sitio en Vivo

El sitio está desplegado en Netlify: [Ver sitio](https://umana-mental-health.netlify.app)

## 📋 Características

- ✅ Diseño responsive (móvil y escritorio)
- ✅ Sección "Quiénes Somos"
- ✅ Servicios de salud mental
- ✅ Horarios de atención
- ✅ Preguntas frecuentes (FAQ)
- ✅ Formulario de contacto
- ✅ Colores y diseño profesional

## 🚀 Despliegue

Este sitio está configurado para desplegarse automáticamente en Netlify. Ver [DEPLOY.md](DEPLOY.md) para instrucciones completas.

### Despliegue Rápido
1. Conecta este repositorio a Netlify
2. Netlify detectará la configuración automáticamente
3. Tu sitio estará en vivo en menos de 1 minuto

## 📁 Estructura del Proyecto

```
Personal/
├── index.html                      # Página principal
├── styles.css                      # Estilos CSS
├── netlify.toml                    # Configuración de Netlify
├── _redirects                      # Reglas de redirección
├── DEPLOY.md                       # Guía de despliegue
├── INSTRUCCIONES_IMAGENES.md       # Guía para agregar imágenes
└── assets/
    └── images/
        └── logo-placeholder.svg    # Logo temporal
```

## 🎨 Personalización

### Agregar tu Logo
Sigue las instrucciones en [INSTRUCCIONES_IMAGENES.md](INSTRUCCIONES_IMAGENES.md)

### Actualizar Información de Contacto

- Teléfono
- Dirección
